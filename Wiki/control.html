MPC Controller

Bullet points: While most papers indicate that human motor control obeys optimality principles, there are some that suggest that a model predictive controller (MPC) might be a more reasonable approximation (link to Wolpert paper). MPCs look ahead several time steps and choose the optimal solution given what the controller can see, rather than finding the true optimal solution.

We use the MPT3 toolbox developed at ETHz, which allows for the simple implementation of MPC. However, it relies on linear models of the system dynamics, so we first linearize the nonlinear plant dynamics of our model using a first-order Taylor series approximation math and then transform the model into discrete time using a zero-order hold approximation math.
